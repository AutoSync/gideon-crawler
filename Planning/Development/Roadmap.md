# Desenvolvimento da dungeon


## Salas

O desenvolvimento de cada sala deve ser feito de forma incremental, começando com a primeira sala e adicionando novas salas conforme o progresso do desenvolvimento.

### Sala 1

A Sala 1 Consiste em ensinar o jogador a se movimentar, atacar e interagir com portas.

- Ensinar movimento
- Ensinar combate simples
- Ensinar interação com porta

Para esta sala/Level, alguns elementos devem ser previamente desenvolvidos, como:


Tiles 
- Paredes Cantos Externos, Paredes Horizontais e Verticais, Junções em T e X
    - [x] 4 Tiles de parede externa L (Prototipo)
    - [x] 2 Tiles de parede horizontal (Prototipo)
    - [x] 2 Tiles de parede externa T (Prototipo)
    - [ ] 1 Tile de parede externa X (Prototipo)
- Piso Formato Quadrado e modular e variavel (para criar salas diferentes)
    - [x] 6 Tile de piso pedra (Prototipo)
    - [ ] 6 Tile de piso pedra com musgo (Prototipo)
    - [ ] 6 Tile de piso pedra quebrada (Prototipo)
- Porta sprite de porta
    - Aberta
        - [ ] 1 Tiles de porta aberta (Prototipo)
    - Fechada
        - [ ] 1 Tiles de porta fechada (Prototipo)

Interação:
- Interação com porta
    - [x] Sistema de interação com porta (Prototipo)
        - [x] Interação para abrir e fechar porta (Prototipo)
        - [ ] Interação para abrir porta com chave (Prototipo)
        - [ ] Interação para abrir porta com puzzle (Prototipo)
        - [ ] Interação para abrir porta com Gemstone (Prototipo)
        - [ ] Interação para abrir porta com selo (Prototipo)
        - [ ] Interação para abrir porta com botão de pressão (Prototipo)
        - [ ] Interação para abrir porta com alavanca (Prototipo)

Personagem
- Personagem principal (Gideon) com animações basicas
    - Movimento
        - [x] Movimentação Basica (Prototipo)
    - Ataque/Dano
        - [ ] Ataque Basico (Prototipo)
    - Inventario
        - [ ] Sistema de inventario (Prototipo)
    - Interação com objetos
        - [ ] Interação com objetos (Prototipo)

Inimigos 
- Inimigo basico com animações basicas
    - Movimento (IA)
    - Ataque/Dano
    - Morte
